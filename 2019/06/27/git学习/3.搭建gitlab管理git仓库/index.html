2. GitLab试用

1. 打开首页

在上面的示例中，我们将宿主主机的8888端口映射到了docker容器的80端口，等待服务启动后，访问192.168.71.104:8888，即可打开gitlab的web页面

2. 设置管理员密码

首先根据提示输入管理员密码，这个密码是管理员用户的密码。对应的用户名是root，用于以管理员身份登录Gitlab,密码设置为了password

3. 创建账号

设置好密码后去注册一个普通账号

4. 创建项目

5. 添加ssh-key到gitlab

项目建好了之后，我们需要将客户端的公钥添加到gitlab，用于免密登陆。

设置地址在右上角用户头像 > Settings > SSH Keys
