基本使用

在 multiprocessing 中，每一个进程都用一个 Process 类来表示。首先看下它的API

  • target 表示调用对象，你可以传入方法的名字
  • args 表示被调用对象的位置参数元组，比如target是函数a，他有两个参数m，n，那么args就传入(m, n)即可
  • kwargs 表示调用对象的字典
  • name 是别名，相当于给这个进程取一个名字
  • group 分组，实际上不使用
